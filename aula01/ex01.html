Mensagem de Alerta

A mensagem de alerta que apareceu ao abrir a página foi feita com JavaScript, usando um script muito simples:

window.alert('Olá, Mundo!')

Pergunta de sim e não

Para criar essa caixa de pergunta, usamos outro script, parecido com o primeiro:

window.confirm('Está gostando de JS?')

Prompt de texto

Para criar esse espaço onde o usuário podo digitar, usamos outro script:

window.prompt('Qual seu nome?')
